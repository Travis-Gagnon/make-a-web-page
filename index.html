Lesson 1

Web Basics

The web consists of four major components:
  • Computer/Browser: What a user, like you, may use to surf the web
  • Internet: A complex network of connected computers
  • Servers: A set of computers that are optimized for file storage and recall
  • HTTP: A language format that is used to express data over the internet; essentially the language of the world wide web

HTML

HTML stands for Hypertest Markup Language and is the primary document type that is used on the internet. HTML is made of up of text (which is the text you see), markup (which is how that text looks on the screen), other documents (such as picture or video), and links to other pages (like this example).

HTML Elements

HTML documents are comprised of HTML Elements, which utilize HTML tags to tell the computer what you wish it to do and how you wish text and documents to be displayed. For instance, you can tell the computer to display the text as bold or italicized just by providing the appropriate tags in the appropriate locations. Tags can also be used to link other webpages or display videos or images (likely of cats).

Inline vs. Block

These tags can be inline or block. Inline tags will just display the contained contents inline with the surrounding text. However, a block element will create an invisible box around the content contained within that tag.

Lesson 2

Creating a More Structured Document

HTML and CSS are both languages used to create webpages. While we learned that HTML is the main language used to add content and structure to a webpage, CSS is a language that controls the style of what is displayed. CSS can change the font size and color, background color, and add borders and margins to displayed files and blocks of text. In fact, CSS can make pages consisting of identical HTML look drastically different from each other.

Tree-like Structure

HTML files are organized into a tree-like structure that makes stylizing the page less redundant with CSS. This structure is referred to as "DOM" or Document Object Model. The tree like structure is a result of HTML elements having other elements within them. One can think of this as elements "branching out" with other sub-divisonal elements much like the trunk of a tree splits into smaller branches (and perhaps again smaller branches).

Internet as a Universe of Boxes

Any internet page can be thought of as a series of boxes that can be arranged in various ways to suit the needs of the developer. These boxes, which are made up of elements like those mentioned above, are more easily organized and arranged then HTML without any coherent organization.

Lesson 3

Adding CSS to Create Style

CSS, or Cascading Style Sheet, is used to apply stylistic effect to an HTML document. The word "cascading" is important because it implies how these CSS rules will shape the final result. CSS will apply to a specific HTML element, but also to all of all that element's child elements as well. This is known as "inheritance" and makes applying sweeping stylistic changes easier.

However, order does matter in CSS. The order with which the rules are applied can affect the outcome. Rules that are stated later in the list will override previous definitions. This allows one to make alter the style of the more specific child elements and negate the "cascading" effect of CSS if they choose.

CSS Selectors

CSS selectors allows a programmer to choose a specific "class" of content that they wish to alter and it will be done more easily, and with less redundancy, than it could be done within the HTML file. A selector could be something like an h3 or div class tag. CSS will then apply a property, such as "font-weight" or "background-color", and a value, such as "bold" or "yellow" respectively, to all elements that match that selector.

Again, this could be done in HTML, but makes the work much more repetitive and time-consuming. Using CSS and structured HTML helps make everything easier.
